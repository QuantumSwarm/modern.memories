# 🧠 Sanity CMS Setup Guide | Unfiltered Fashion

This guide walks you through initializing Sanity.io within your Next.js 14 project. Sanity will serve as the headless CMS for managing designers, collections, and event details.

## 🛠 Prerequisites
- Node.js (v18+)
- npm or yarn
- VS Code
- A Sanity Account (free tier is sufficient)

## 🚀 Installation Steps

1. **Initialize Sanity**
   Open your VS Code terminal and run:
   ```bash
   npm create sanity@latest
